Perhatian!

1. 	Ubuntu 8.10 only. Di atas Ubuntu ini akan membuat paket-paket tidak bekerja atau tidak dapat dipanggil. Ini dikarenakan Ubuntu/Debian terbaru sudah memiliki paket-paket yang saya sediakan dengan versi yang lebih baru. Misal paket gcc yang saya berikan memiliki versi 3.4 dan di tahun ini (2020) memiliki versi 10. Sehingga apabila kita mengeksekusi perintah gcc, maka yang akan terpanggil hanya versi terbaru (10 atau lebih seiring rilisnya versi terbaru).

2. 	Link Ubuntu 8.10
	1. 	http://old-releases.ubuntu.com/releases/8.10/ubuntu-8.10-desktop-i386.iso

3. 	Tidak disarankan menggunakan cygwin atau instalasi langsung ke harddisk. Sebaiknya gunakan Virtual Box. Virtual Box menyediakan virtual machine yang dapat membantu kita menginstal OS apapun secara virtual, baik Windows ataupun Linux. Kekurangannya, virtual mesin harus berbagi kinerja dengan OS induk. Jadi, jika ingin menggunakan virtual mesin di sarankan untuk tidak mengeksekusi program lain yang cukup berat seperti browser ataupun aplikasi editing.

4.	(Warning!) Bila ingin menggunakan Ubuntu 8.10 sebagai salah satu OS utama (Install harddisk), diharapkan telah memahami pengaturan partisi harddisk. Bila tidak, rekan-rekan akan kehilangan partisi C atau D windows kalian (Local Disk C atau DATA D terhapus). 

5.	Selain itu, saya sengaja memilih versi ini agar rekan-rekan dapat menggunakannya bersama sistem windows kalian yang di jalankan via VirtualBox. 

6.	Panduan pengaturan partisi, ubuntu, ataupun virtualbox, bisa di cari di Youtube dan web lain. Mengenai panduan partisi, sebaiknya rekan-rekan mempelajari juga tipe-tipe partisi utama seperti FAT dan NTFS untuk windows, serta EXT,BTRFS, dll untuk lilnux.
